3. Componenti UI Principali
NextMatchCard
Card "Hero" per il prossimo evento.

Sfondo: Linear Gradient ['#22c55e', '#16a34a'].
Shadow: Colore #22c55e, Offset {0,8}, Radius 16, Opacity 0.3.
Border Radius: 24.
Micro-interactions: BlurView (intensity 30) sui loghi dei team.
CalendarPicker
Calendario interattivo per selezione date.

Stati Giorno:
Today: Border 1px Primary.
Selected: Border 2px Primary (cerchio vuoto).
Event (Future): Sfondo Solid Green (#2E7D32) + Testo Bianco.
Event (Past): Sfondo Solid Orange (#f97316) + Testo Bianco.
Layout: Grid 7 colonne, Aspect Ratio 1:1.
MatchMap
Card visualizzazione mappa evento.

Container: Border Radius 16, Border 1px (theme.border).
Map View: Altezza fissa 160px.
Marker: Pin colore primario (theme.primary).
Avatar
Elemento circolare fallback per immagini profilo.

Logica: Genera colore di sfondo deterministico/random.
Contenuto: Iniziali del nome (max 2 caratteri), testo bianco bold.
4. Elementi Interattivi (Bottoni)
I bottoni non sono centralizzati ma seguono un pattern comune (es. in 
MatchMap
).

Primary Button

Container: borderRadius: 12, padding: 12.
Colore: Primary Theme Color.
Shadow: Offset {0,4}, Opacity 0.3, Radius 8.
Contenuto: Icona (opzionale) + Testo (White, Bold, Size 14).
5. Struttura Navigazione (Tabs)
La Bottom Tab Bar è personalizzata in 
app/(tabs)/_layout.tsx
.

Altezza: 88px (iOS), 60px (Android).
Icone: MaterialIcons (size 28).
Elevation: 8 (Android) per ombra, 0 per iOS (gestito nativamente).
